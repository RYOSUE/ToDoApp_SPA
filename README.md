# Next.js + React ToDoApp
このページに飛んでいただいた皆様に心より感謝申し上げます。  

ここで作成しているのはNext.jsとReactを使ったシングルページのToDoアプリです。  
認証・権限管理・レスポンシブデザインなど実用的な機能を備える予定です。

## 技術スタック

- TypeScript
- React
- Next.js
- Bootstrap CSS
- Jest（テスト）
- MySQL（データベース）

## セットアップ

1. リポジトリをクローン
2. 依存パッケージをインストール  
   `npm install`
3. 環境変数を設定
4. 開発サーバー起動  
   `npm run dev`
